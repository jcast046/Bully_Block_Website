Interface

The BullyBlock system provides an intuitive and user-friendly interface with the following components:

  • Dashboard: Displays cyberbullying incidents and alerts.
  • Incident Review Panel: Allows administrators to review flagged messages.
  • Settings: Configuration options for detection sensitivity and notifications.
  • Reports: Provides analytical insights on cyberbullying trends.
