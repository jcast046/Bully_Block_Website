Getting Started

Requirements

Access the BullyBlock GitHub Repository to get started

Node.js and Express

  • Node.js (version 18 or later)
  • npm (Node Package Manager)

Terminal Access

  • Command line interface (CLI)
  • Basic knowledge of terminal commands

Web Browser

  • Google Chrome (latest version)
  • Mozilla Firefox (latest version)
  • Microsoft Edge (latest version)
  • Safari (latest version)

Internet Connection

  • Stable broadband connection

Running the Application

1

Downloading

  1. Clone the repository
  2. Using a terminal, navigate to the project directory
2

Starting from the Frontend

  1. cd into bullyblock-dashboard
  2. Install dependencies using npm install
  3. Run the command npm start in the terminal
3

Starting from the Backend

  1. Ensure that Node.js and npm are installed
  2. Using a terminal, navigate to the "Backend" directory
  3. Run npm install to ensure dependencies such as Express, Mongoose, and Axios are installed
  4. Run npm start in the terminal
